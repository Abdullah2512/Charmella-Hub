# 💎 CharmellaHub

CharmellaHub is a modern full-stack jewelry e-commerce web application built with ASP.NET Core MVC. It provides a seamless shopping experience with secure authentication, online payments, product management, and an intuitive user interface.

## ✨ Features

- 🔐 User Authentication & Authorization
- 🛍️ Browse Products by Categories
- ❤️ Wishlist Management
- 🛒 Shopping Cart
- 💳 Secure Stripe Payment Integration
- 📧 Email Notifications
- 👨‍💼 Admin Dashboard
- 📦 Product & Inventory Management
- 📱 Responsive UI
- 🔍 Product Search
- 👤 User Profile Management
- 🔒 Session Management

## 🛠 Tech Stack

**Frontend**
- HTML5
- CSS3
- JavaScript
- Bootstrap

**Backend**
- ASP.NET Core MVC (.NET 10)
- C#

**Database**
- SQL Server
- Entity Framework Core

**Other Technologies**
- Stripe Payment Gateway
- MailKit (Email Service)
- Session Authentication
